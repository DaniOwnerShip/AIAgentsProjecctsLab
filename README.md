# AI Agents Projects Lab

Repositorio de experimentación con **agentes de inteligencia artificial** y desarrollo asistido por IA.

Este laboratorio contiene pequeños proyectos y pruebas creados mediante **prompt-driven development**, utilizando herramientas de programación asistida por IA.

## Contenido

El repositorio está organizado en carpetas según la herramienta utilizada:

- **Cursor/**  
  Proyectos experimentales desarrollados utilizando Cursor AI mediante prompts.

- **ClaudeCode/**  
  Pequeños experimentos generados con Claude Code.

## Objetivo

Explorar nuevas formas de desarrollo de software mediante:

- Agentes de inteligencia artificial
- Prompt engineering
- Desarrollo asistido por IA
- Prototipado rápido de ideas

Los proyectos incluidos son **pruebas conceptuales y experimentos técnicos**, enfocados en aprender y explorar las capacidades actuales de las herramientas de programación con IA.
